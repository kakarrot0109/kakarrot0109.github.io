VsCode配置备份

作者: KAKARROT   日期: 2024年12月23日   分类: VsCode

关键配置解释：

  • Python 相关： 针对 Python 开发配置了解释器路径、格式化工具和代码检查工具，提高开发效率和代码质量。
  • Markdown 相关： 优化了 Markdown 预览和格式化，方便写作和 Hexo 博客编辑。
  • 编辑器通用设置： 调整了字体、行高、缩进等，提高阅读和编辑的舒适度。
  • 工作台设置： 设置了主题、图标主题等，美化 VS Code 界面。
  • 代码格式化: 保存时自动整理 import 顺序，保证代码风格一致

关键快捷键解释：

  • ctrl+shift+f：全局搜索文件，快速定位文件。
  • ctrl+alt+n：新建文件，提高创建文件的效率。
  • ctrl+alt+w：关闭当前标签页，方便管理多个文件。
  • ctrl+alt+left/right：切换标签页，快速在不同文件之间切换。
  • ctrl+d: 快速复制当前行
  • ctrl+shift+d: 快速复制当前行并粘贴到上一行
  • alt+up/down: 快速上下移动当前行

如何使用：

  1. 打开 VS Code，按下 Ctrl + Shift + P (Windows) 或 Cmd + Shift + P (Mac)。
  2. 输入 “Preferences: Open Settings (JSON)”，打开 s”ettings.json” 文件。
  3. 复制上面的 “settings.json” 代码，粘贴到文件中并保存。
  4. 同样的方式打开 “Preferences: Open Keyboard Shortcuts (JSON)”，打开 “keybindings.json” 文件。
  5. 复制上面的 “keybindings.json” 代码，粘贴到文件中并保存。
